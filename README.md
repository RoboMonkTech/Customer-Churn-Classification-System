# 🚀 Customer Churn Classification System for Business Retention Analytics

An AI-powered Machine Learning web application that predicts whether a customer is likely to **continue using a company's services** or **churn (leave the company)**. The system uses a **Random Forest Classifier** trained on the Telco Customer Churn Dataset and provides instant predictions through an interactive web interface.

---

## 📌 Project Overview

Customer churn prediction is an important business analytics task that helps organizations identify customers who are likely to discontinue their services. This project applies Machine Learning techniques to analyze customer information and classify customers into:

- ✅ Retained
- ❌ Likely to Churn

The application assists businesses in improving customer retention, reducing customer loss, and making better business decisions.
